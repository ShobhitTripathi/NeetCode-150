#  Generate Parentheses

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

Example 1:
```
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```
Example 2:
```
Input: n = 1
Output: ["()"]
 ```

Constraints:
```
1 <= n <= 8
```
Algorithm
```
Start with an empty string.
Track:
  open → number of '(' used
  close → number of ')' used
  If open == close == n, add the built string to the result.
  If open < n, add '(' and recurse.
  If close < open, add ')' and recurse.
  Backtrack after each choice.
```

Time & Space Complexity
```
Time complexity: O(Cₙ) where Cₙ is the n-th Catalan number
Space complexity: O(n)
```

```java
class Solution {

    // Stores all valid combinations of parentheses
    private List<String> result;

    public List<String> generateParenthesis(int n) {
        result = new ArrayList<>();

        // StringBuilder is used to efficiently build the current combination
        StringBuilder sb = new StringBuilder();

        // Start backtracking with:
        // openN = number of '(' used so far
        // closedN = number of ')' used so far
        backtrack(0, 0, n, sb);

        return result;
    }

    private void backtrack(int openN, int closedN, int n, StringBuilder sb) {

        // Base case:
        // When both open and close counts reach n,
        // we have a complete valid parentheses string
        if (openN == n && closedN == n) {
            result.add(sb.toString());
            return;
        }

        // Choice 1: Add '(' if we still have opening brackets left
        if (openN < n) {
            sb.append('(');
            backtrack(openN + 1, closedN, n, sb);
            // Backtrack: remove last added '('
            sb.deleteCharAt(sb.length() - 1);
        }

        // Choice 2: Add ')' only if it keeps the string valid
        // i.e., number of ')' never exceeds '('
        if (closedN < openN) {
            sb.append(')');
            backtrack(openN, closedN + 1, n, sb);
            // Backtrack: remove last added ')'
            sb.deleteCharAt(sb.length() - 1);
        }
    }
}
```
