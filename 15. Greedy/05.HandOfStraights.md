# [846. Hand of Straights](https://leetcode.com/problems/hand-of-straights/)
Medium


Alice has some number of cards and she wants to rearrange the cards into groups so that each group is of size groupSize, and consists of groupSize consecutive cards.

Given an integer array hand where hand[i] is the value written on the ith card and an integer groupSize, return true if she can rearrange the cards, or false otherwise.

 

Example 1:
```
Input: hand = [1,2,3,6,2,3,4,7,8], groupSize = 3
Output: true
Explanation: Alice's hand can be rearranged as [1,2,3],[2,3,4],[6,7,8]
```
Example 2:
```
Input: hand = [1,2,3,4,5], groupSize = 4
Output: false
Explanation: Alice's hand can not be rearranged into groups of 4.
```
 

Constraints:

- 1 <= hand.length <= 104
- 0 <= hand[i] <= 109
- 1 <= groupSize <= hand.length
 

> Note: This question is the same as 1296: https://leetcode.com/problems/divide-array-in-sets-of-k-consecutive-numbers/

## Approach
```
1: Always start forming groups from the smallest available card and greedily build consecutive sequences of size groupSize. // O(NlogN)

2: For each number, expand backward to find the smallest possible starting point and greedily form full consecutive groups of size groupSize. // O(N)
```

## Solution
```java
// O(NlogN) : O(N)
public class Solution {
    public boolean isNStraightHand(int[] hand, int groupSize) {

        // If total cards can't be divided into equal groups → impossible
        if (hand.length % groupSize != 0)
            return false;

        // Frequency map of all card values
        Map<Integer, Integer> count = new HashMap<>();
        for (int n : hand)
            count.put(n, count.getOrDefault(n, 0) + 1);

        // Min-heap to always pick the smallest available card to start a group
        PriorityQueue<Integer> minH = new PriorityQueue<>(count.keySet());

        // Try forming groups starting from the smallest available card
        while (!minH.isEmpty()) {
            int first = minH.peek();  // smallest available card value

            // Attempt to form a consecutive group: first, first+1, ..., first+groupSize-1
            for (int i = first; i < first + groupSize; i++) {

                // If any required card is missing → fail
                if (!count.containsKey(i))
                    return false;

                // Use one copy of this card
                count.put(i, count.get(i) - 1);

                // If count becomes zero, this card must be the current smallest
                if (count.get(i) == 0) {
                    if (i != minH.peek())  // can't skip gaps
                        return false;
                    minH.poll();           // remove from heap
                }
            }
        }

        return true;
    }
}



// 2:  O(N) : O(N)

public class Solution {
    public boolean isNStraightHand(int[] hand, int groupSize) {
        // If total cards can’t be evenly divided into groups → impossible
        if (hand.length % groupSize != 0) return false;

        // Count frequency of each card
        Map<Integer, Integer> count = new HashMap<>();
        for (int num : hand) {
            count.put(num, count.getOrDefault(num, 0) + 1);
        }

        // For each card, try to find the earliest consecutive group it belongs to
        for (int num : hand) {

            int start = num;

            // Move start backward to the smallest possible number for this group
            while (count.getOrDefault(start - 1, 0) > 0) start--;

            // Move forward from the computed start to 'num'
            while (start <= num) {

                // While this 'start' value still exists, try forming a full group
                while (count.getOrDefault(start, 0) > 0) {

                    // Build a consecutive sequence of length groupSize
                    for (int i = start; i < start + groupSize; i++) {
                        if (count.getOrDefault(i, 0) == 0) return false;
                        count.put(i, count.get(i) - 1);
                    }
                }

                start++; // increment the start to check next possible sequence
            }
        }

        return true;
    }
}
```
